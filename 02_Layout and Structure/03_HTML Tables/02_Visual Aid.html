Example 3: Column Spanning (colspan)

The yellow cell spans across 3 columns using colspan="3":

Monday Tuesday Wednesday
Team Meeting (All Days)
Work Study Project

Note: When a cell spans 3 columns, we only have one <td> in that row!

Example 4: Row Spanning (rowspan)

The green cell spans down 3 rows using rowspan="3":

Time Activity Location
Morning
9AM-12PM		 Coding Lab A
Testing Lab B
Review Room 101

Important: The rows that are spanned don't include that cell!
