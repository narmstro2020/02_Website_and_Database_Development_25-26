CSS Syntax Breakdown

h1 {
    color: blue;
    font-size: 36px;
}

Components:

  • Selector (h1) - What to style
  • Property (color, font-size) - What aspect to change
  • Value (blue, 36px) - What to change it to

Three Ways to Add CSS

Method 1: Inline CSS
<p style="color: red; font-size: 20px;">This is inline CSS</p>

This is inline CSS

Use for quick tests or when you need to override other styles
Method 2: Internal CSS
<style>
  p { color: blue; font-size: 18px; }
</style>

This is internal CSS

Good for single-page styling or page-specific styles
Method 3: External CSS
<link rel="stylesheet" href="styles.css">

/* In styles.css */
p { color: green; font-size: 16px; }

This is external CSS

Best for multi-page websites - write once, use everywhere!

CSS Cascade Priority

When multiple styles conflict, CSS follows this priority:

  • 1
    Inline CSS (Highest Priority)
    style="color: red;"
  • 2
    Internal CSS (Medium Priority)
    <style> p { color: blue; } </style>
  • 3
    External CSS (Lowest Priority)
    In separate .css file
If the same priority level has multiple rules, the last one wins!
