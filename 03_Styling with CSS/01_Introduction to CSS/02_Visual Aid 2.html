Understanding HSL:

Hue: The color position on the wheel (0-360°)

Saturation: How vivid the color is (0% = gray, 100% = pure color)

Lightness: How light or dark (0% = black, 50% = normal, 100% = white)

RGB Color Mixing

Combining Red, Green, and Blue creates all colors

Hex Tips:

• Shorthand: #RGB becomes #RRGGBB (e.g., #F63 = #FF6633)

• Values range from 00 (none) to FF (maximum)

• Web safe colors use 00, 33, 66, 99, CC, FF

Alpha Channel:

• 0 = Fully transparent (invisible)

• 0.5 = Semi-transparent (50% see-through)

• 1 = Fully opaque (solid)
