Absolute Units Reference:

  • px - Pixels (1px = 1/96th of 1 inch on screens)
  • in - Inches (1in = 96px)
  • cm - Centimeters (1cm = 37.8px)
  • mm - Millimeters (1mm = 3.78px)
  • pt - Points (1pt = 1.33px)
  • pc - Picas (1pc = 16px)

Em vs Rem - The Key Difference

Em Units (Cascading)

Parent: 16px base
Child: 1.5em = 24px (1.5 × 16)
Grandchild: 1.2em = 28.8px (1.2 × 24)

⚠️ Em values compound through nesting!

Rem Units (Root-based)

Parent: Any size
Child: 1.5rem = 24px (1.5 × 16 root)
Grandchild: 1.2rem = 19.2px (1.2 × 16 root)

✓ Rem always references the root size!

💡 Pro Tips:

  • Use rem for consistent spacing and typography across your site
  • Use em for properties that should scale with the component's font size
  • Use % for flexible layouts that adapt to parent containers
  • Use px only for small, fixed values like borders
  • Use viewport units sparingly for truly responsive elements
