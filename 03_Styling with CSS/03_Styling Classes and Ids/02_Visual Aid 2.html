🎨 CSS Selectors Visual Guide

Element Selector

p {
  color: blue;
}
Specificity: 1 point
Selects: ALL paragraph elements
<p>This is selected</p>
<p>This too!</p>

Class Selector

.highlight {
  background: yellow;
}
Specificity: 10 points
Selects: Elements with class="highlight"
<p class="highlight">Selected!</p>
<div class="highlight">Also selected!</div>

ID Selector

#header {
  font-size: 24px;
}
Specificity: 100 points
Selects: ONE element with id="header"
<div id="header">Only this!</div>
Feature Class (.classname) ID (#idname)
Can be reused? ✓ Yes, unlimited times ✗ No, must be unique
Multiple per element? ✓ Yes (space-separated) ✗ No, only one ID
Specificity weight 10 points 100 points (wins!)
Best for Reusable styles, groups Unique elements, anchors
CSS symbol Period (.) Hash (#)

Grouping (OR)

h1, h2, h3 {
  color: navy;
}
Selects: h1 OR h2 OR h3 elements
<h1>Selected</h1> ✓
<h2>Selected</h2> ✓
<h3>Selected</h3> ✓

Chaining (AND)

.warning.text {
  color: red;
}
Selects: Elements with BOTH classes
<p class="warning">Not selected</p> ✗
<p class="warning text">Selected!</p> ✓
