Content: The actual content area where text and images appear. This is what width and height properties control.

Padding: Transparent space around the content, inside the border. Background color extends into padding.

Border: A visible line that goes around the padding and content. Has its own color and style.

Margin: Transparent space outside the border. Creates gaps between elements. Background color doesn't extend here.
